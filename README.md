# Sistema de Gestión de Tareas Colaborativas

## Descripción

Aplicación web desarrollada con Flask y Firebase para gestionar proyectos, tareas y usuarios en equipos de trabajo. Permite crear, consultar, editar y eliminar proyectos y tareas, además de registrar usuarios y realizar autenticación mediante inicio de sesión.

## Tecnologías Utilizadas

* Python
* Flask
* Firebase Firestore
* HTML
* CSS
* Git y GitHub

## Funcionalidades

### Gestión de Proyectos

* Crear proyectos
* Consultar proyectos
* Editar proyectos
* Eliminar proyectos

### Gestión de Tareas

* Crear tareas
* Editar tareas
* Eliminar tareas
* Asignar tareas a usuarios
* Definir prioridad
* Definir fecha límite
* Filtrar tareas por estado

### Gestión de Usuarios

* Registro de usuarios
* Inicio de sesión
* Listado de usuarios

### Dashboard

* Total de proyectos
* Total de tareas
* Tareas pendientes
* Tareas en progreso
* Proyectos finalizados

## Base de Datos

Firebase Firestore

Colecciones:

* usuarios
* proyectos
* tareas

## Instalación

1. Clonar el repositorio
2. Crear entorno virtual
3. Instalar dependencias

pip install flask firebase-admin

4. Ejecutar aplicación

python app.py
